NAME

netl_generate_c , - generate a netl module from a netl config file.

SYNOPSIS

#include 
void  netl_generate_c (FILE * fp );

DESCRIPTION

after running the netl (8) configuration routines (sa), this routine can be run to generate an equivalent netl_module (1) . fp should be opened for writ ing, which is where the c program will be generated. netl_generate_c will close fp when it is done.

SEE ALSO

netl (8) , netl_api (3) , netl_module (1)
