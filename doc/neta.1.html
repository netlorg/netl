NAME

neta - network packet analyzer

SYNOPSIS

neta [options] file [...]

DESCRIPTION

neta is a network packet analyzer. it was designed to decode an IP packet stored as a file. usually this will be a packet dumped by netl (8) . first the address is shown, followed by header data. the final part is a hex dump, which displays offset, hex data, and character data. because the header data is interpreted for you, it is not included in the hex dump.

OPTIONS

neta uses a subset of the options from netl (8) . these include -h, -r and -v. the reminder do not make sense.
